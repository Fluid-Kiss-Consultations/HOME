## Brand Identity Decisions

### Naming

The legal entity is **Fluid Kiss Consultations (FKC)**. The domain is **fluidsolutions.online** and the site currently displays "Fluid Solutions." These need to be reconciled.

**Recommendation:** Use "Fluid Kiss Consultations" as the primary brand name on the site. "Fluid Solutions" can remain as the domain — it's shorter, memorable, and the `.online` TLD already signals modernity. The logo and all page content should reference FKC. The legal footer uses the full name.

### Positioning

**Systems Oracle — Full-Service Consulting**

The site must communicate: FKC doesn't consult in silos. FKC sees entire systems — organizational, technical, economic, cultural — diagnoses where they're broken, and fixes them from the intersection of every discipline involved.

### Motto (Canon)

**Innovate. Elevate. Inspire.**

### Brand Voice

- Direct, confident, zero-jargon
- Speaks to decision-makers, not procurement departments
- Doesn't beg for attention — states what it does and why it matters
- Warmth without softness — the tone of someone who's built the thing, run the thing, and knows why it breaks

## Accessibility Compliance Checklist

From your research (Strategic Framework for Organizational Digital Accessibility):

- [ ] High contrast text-to-background ratio across all sections
- [ ] Scalable typography — fonts adapt when user enlarges text
- [ ] No flickering graphics (nothing above 3 flashes/sec)
- [ ] Alt-text on every image (logo, photos, section backgrounds)
- [ ] Flat hierarchy — all content reachable within 1–3 clicks
- [ ] Consistent navigation across all pages
- [ ] Plain English — short sentences, active voice, subject-verb proximity
- [ ] Touch targets large enough for thumb navigation on mobile
- [ ] SSL certificate active (should be default with IONOS)
- [ ] Custom 404 error page
